# plan de réalisation du projet final de developpement d'application desktop

## contexte du projet:
Dans le but d'éffectuer la transformation digitale d'une assurance, nous sommes amenés à créer leur outil interne de gestion des contrats des assurés.  

## Objectif:
A la fin de ce projet nous aurons développé une application desktop permettant après s'être authentifier gérer(ajouter, modifier, supprimer) les contrats des assurés avec l'assurance

## Etapes pour la réalisation du projet:
L'oganisation pour la réalisation de ce projet va se dérouler comme suit: 
Dans un 1er temps nous allons concevoir le projet ensuite on passe à la rédaction du code suivi des tests de l'application pour terminer avec la création des exécutables et des autres fichiers demandés.  
Voici les différentes étapes pour réaliser ce projet :  

### étapes de la phase de conception:
1. mettre en place les maquettes de l'interface utilisateur;
2. concevoir la base de données(tables et fichiers de base de données sqlite).

### étapes de codage:
1. créer le projet javafx et y ajouter les fichiers fxml pour chaque interface demandé( page de connexion (Login), page d'inscription (Sign Up), page d'accueil (Home) affichant un tableau de données (TableView) et quelques informations, et page de formulaire d'ajout);
2. coder l'interface des pages dans l'ordre suivant les maquettes;
3. tests et validations;
4. création de l'éxécutable et de l'installeur.


### rédaction des documents : 
1. Rédaction du fichier README permettant de reproduire la compilation et l'exécution (prérequis, étapes, identifiants de test);
2. Rédaction de la documentation (guide d'installation et d'utilisation, captures des quatre écrans);
3. réalisation de la vidéo.
